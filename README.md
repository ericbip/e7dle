# E7dle ⚔️

Wordle-style guessing game for **Epic Seven** — guess the daily hero from their attributes.

🎮 **[Play here](https://e7dle.vercel.app)**

## How to play

Type a hero name, pick from the dropdown, and use the colored tiles as hints:
- 🟩 **Green** — exact match
- ⬛ **Grey** — no match

Attributes shown: Element · Role · Zodiac · Gender · Rarity

## Stack

Vanilla HTML + CSS + JS. No framework, no build step. Deploys as a static site.
